# 项目 4：Agent 平台项目

## 目标

做一个最小可运行的 Agent 任务执行平台，重点展示平台开发能力：工具注册、任务状态机、日志追踪、人工确认、失败重试、成本统计和权限控制。

## 必备功能

- 工具注册。
- Agent 任务执行。
- 任务状态机。
- 日志追踪。
- 人工确认。
- 失败重试。
- 成本统计。
- 权限控制。

## 建议场景

- 自动整理文档并生成摘要。
- 根据需求生成接口草稿。
- 调用搜索、数据库查询、文件读写等受控工具。

## 里程碑

1. 工具注册表和工具调用接口。
2. 任务创建和状态机。
3. 执行日志追踪。
4. 人工确认节点。
5. 失败重试和超时控制。
6. 成本统计。
7. 权限控制和审计日志。
8. README 展示一次任务执行全过程。

## 面试证明材料

- 状态机图。
- 任务日志样例。
- 工具权限设计。
- 失败重试策略。
- 成本统计字段说明。
