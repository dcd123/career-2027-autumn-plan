# 项目 3：RAG 知识库项目

## 目标

做一个能展示 AI 应用工程能力的 RAG 知识库系统。重点不是接一个模型接口，而是把文档处理、检索质量、引用溯源和评测做成可解释的工程链路。

## 必备功能

- 文档上传。
- 文档解析。
- 文本切分。
- 向量化。
- 检索。
- 重排。
- 引用溯源。
- 问答评测集。

## 建议技术栈

- Java/Spring Boot 作为服务端。
- PostgreSQL + pgvector 或 Milvus。
- 对象存储可先用本地文件系统替代。
- 可接入任意可用 embedding 和 chat model。

## 里程碑

1. 文档上传和元数据存储。
2. PDF/Markdown/TXT 解析。
3. 文本切分和 chunk 元数据设计。
4. 向量化入库。
5. 检索接口。
6. 重排和引用溯源。
7. 构造 30 条问答评测集。
8. README 展示一次完整问答链路。

## 面试证明材料
