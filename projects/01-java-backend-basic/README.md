# 项目 1：Java 后端基础项目

## 目标

做一个可部署、可演示、可面试讲述的后端基础项目，重点证明 Spring Boot、MySQL、Redis、权限、接口规范和工程化能力。

## 必备功能

- 登录注册。
- RBAC 权限。
- MySQL 表设计。
- Redis 缓存或登录态支持。
- RESTful API。
- 统一异常处理。
- 参数校验。
- 接口文档。
- Docker 部署。

## 建议技术栈

- Java 17。
- Spring Boot 3。
- MySQL 8。
- Redis 7。
- MyBatis Plus 或 Spring Data JPA。
- Knife4j / Swagger。
- Docker Compose。

## 里程碑

1. 项目初始化、分层结构、健康检查接口。
2. 用户注册登录、密码加密、参数校验。
3. RBAC 表设计和鉴权流程。
4. Redis 接入。
5. 统一异常、日志、接口文档。
6. Docker Compose 部署。
7. README、接口截图、面试问答整理。

## 面试证明材料

- ER 图。
- 接口文档截图。
- 核心鉴权流程图。
- Docker 启动说明。
- 3 个关键代码片段。
