# 简历项目 Bullet 草稿

## 写法规则

- 写清楚业务场景、负责模块、技术方案、结果。
- 不堆技术名词。
- 每个 bullet 尽量能被项目 README、代码或压测记录支撑。

## 项目 1：Java 后端基础项目

- 基于 Spring Boot + MySQL + Redis 实现用户与权限管理系统，负责登录注册、RBAC 鉴权、统一异常处理和接口文档，支持 Docker Compose 一键启动。

## 项目 2：高并发系统项目

- 基于 Redis 缓存、MQ 异步削峰和接口限流实现高并发业务流程，负责幂等校验、分布式锁和索引优化，并输出压测前后对比报告。

## 项目 3：RAG 知识库项目

- 构建支持文档上传、解析、向量检索、重排和引用溯源的知识库问答系统，设计问答评测集用于记录召回与回答质量。

## 项目 4：Agent 平台项目

- 设计 Agent 任务执行平台，支持工具注册、任务状态机、日志追踪、人工确认、失败重试、成本统计和权限控制。
