# 2026-W08：阶段前半程收束与可展示产出

日期：2026-08-10 至 2026-08-16

## 1. 本周唯一主目标

把前 7 周产出整理成可检查、可复述、可继续迭代的阶段包。

## 2. 本周任务

### 必做任务

- 整理前 7 周算法记录，标记 20 道高价值错题。
- 整理 Java 基础、集合、IO、泛型、并发、Maven/JUnit 笔记索引。
- 给 Maven 小项目补齐 README、测试说明和下一阶段 TODO。
- 完成 10 道错题二刷。
- 完成 1 次阶段复盘和 1 次 Git commit。

### 选做任务

- 为项目画一张简单模块图。
- 录一次 10 分钟项目讲解音频或写文字稿。

### 低状态保底任务

- 完成 3 道错题二刷。
- 写 1 篇短笔记：前 8 周最卡的一个点。
- 做 1 次 README 或测试 commit。
- 完成 1 次周复盘。

## 3. 时间安排

总计：28 小时。

- 周一：4 小时
  - 汇总算法记录，1.5 小时，输出错题清单。
  - 二刷 2 道错题，1.5 小时，输出二刷记录。
  - 更新算法追踪表，1 小时，输出状态标记。
- 周二：4 小时
  - 整理 Java 笔记索引，1.5 小时，输出 notes 索引。
  - 二刷 2 道错题，1.5 小时，输出二刷记录。
  - 标记 5 个仍需复述的问题，1 小时，输出面试追踪记录。
- 周三：4 小时
  - 更新 Maven 小项目 README，2 小时，输出运行、测试、功能说明。
  - 二刷 1 道错题，1 小时，输出二刷记录。
  - 运行测试并记录结果，1 小时，输出测试记录。
- 周四：4 小时
  - 整理项目 TODO 和下一阶段接 Spring Boot 的接口想法，1.5 小时，输出 TODO。
  - 二刷 2 道错题，1.5 小时，输出二刷记录。
  - 写阶段复盘大纲，1 小时，输出初稿。
- 周五：4 小时
  - 二刷 3 道错题，2 小时，输出二刷记录。
  - 完成阶段复盘，1 小时，输出复盘文件。
  - 整理本周 5 个面试问题答案，1 小时，输出追踪记录。
- 周六：5 小时
  - 给项目补 2 个测试或小修复，2 小时，输出测试或代码。
  - 写项目讲解稿，1.5 小时，输出文字稿。
  - 提交 Git commit，1.5 小时，输出提交记录。
- 周日：3 小时
  - 复述项目功能和 Java 基础知识，1 小时，输出文字稿。
  - 检查前 8 周文件是否齐全，1 小时，输出缺口清单。
  - 调整 W09-W12 滚动目标，1 小时，输出下一轮计划建议。

## 5. 输出物

- 至少 10 道错题二刷记录。
- 1 份前 8 周算法错题清单。
- 1 份 notes 索引或目录说明。
- 1 个 Maven 小项目 README 更新。
- 1 篇阶段复盘。
- 1 次 Git commit。

## 6. 验收标准

- 能指出前 8 周最重要的 20 道错题，并说明其中 5 道的错因。
- 能通过 README 运行 Maven 小项目和测试。
- 能用 5 分钟讲清项目当前功能、代码结构、下一步计划。
- 能复述 Java 基础、集合、IO、泛型、并发各 1 个高频问题。
- 能列出 W09-W12 只推进一个主线的滚动计划建议。
