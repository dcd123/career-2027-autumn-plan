# 2026-W05：泛型、枚举与二分题型

日期：2026-07-20 至 2026-07-26

## 1. 本周唯一主目标

用泛型和枚举改造已有 Java 小项目，并完成二分题型第一轮训练。

## 2. 本周任务

### 必做任务

- 写泛型类、泛型方法、枚举、Optional 基础练习。
- 用泛型改造命令行项目中的数据容器或结果返回结构。
- 完成 10 道二分、数组边界、查找题。
- 写 1 篇笔记：泛型、类型擦除和常见使用场景。
- 完成 1 次 Git commit 和 1 次周复盘。

### 选做任务

- 阅读 JavaGuide 关于反射和注解的入门内容。
- 写一个统一 Result 返回对象练习。

### 低状态保底任务

- 完成 3 道二分题。
- 写 1 篇短笔记：泛型解决了什么问题。
- 做 1 次项目 commit。
- 完成 1 次周复盘。

## 4. 学习输入

- JavaGuide：Java 泛型、Optional、枚举相关内容。
- Oracle Java Tutorials：Generics。

- LeetCode：二分查找、搜索插入位置、在排序数组中查找元素第一个和最后一个位置、寻找旋转排序数组中的最小值。
