# 2026-W02：Java 面向对象与链表基础

日期：2026-06-29 至 2026-07-05

## 1. 本周唯一主目标

用 Java 写出面向对象练习，并完成链表题的第一轮训练。

## 2. 本周任务

### 必做任务

- 写类、封装、继承、接口、多态、异常处理相关练习。
- 完成 10 道链表与数组综合题。
- 写 1 篇笔记：接口、抽象类、继承的使用边界。
- 给命令行计算器补充异常处理和输入校验。
- 完成 1 次 Git commit 和 1 次周复盘。

### 选做任务

- 给计算器增加历史记录功能。
- 整理 Java 常见异常分类表。

### 低状态保底任务

- 完成 3 道链表题。
- 写 1 篇短笔记：接口和抽象类区别。
- 做 1 次项目 README 或代码 commit。
- 完成 1 次周复盘。
