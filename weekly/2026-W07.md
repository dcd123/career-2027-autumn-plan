# 2026-W07：Maven、JUnit 与项目雏形

日期：2026-08-03 至 2026-08-09

## 1. 本周唯一主目标

把零散 Java 练习整理成一个可测试的 Maven 小项目雏形。

## 2. 本周任务

### 必做任务

- 创建或整理 Maven 项目结构：src/main/java、src/test/java、pom.xml。
- 给已有命令行模块补充 JUnit 测试。
- 完成 10 道算法复习题，重点复刷前 6 周错题。
- 写 1 篇笔记：Maven 坐标、依赖、生命周期、测试命令。
- 更新项目 README：运行方式、测试方式、当前功能。

### 选做任务

- 加入 Checkstyle 或简单格式化说明。
- 给项目拆分 service、model、util 包。

### 低状态保底任务

- 完成 3 道错题复刷。
- 写 1 篇短笔记：Maven 生命周期。
- 做 1 次测试或 README commit。
- 完成 1 次周复盘。

## 3. 时间安排

总计：29 小时。

- 周一：4 小时
  - 阅读 Maven 基础，1 小时，输出要点清单。
  - 创建 Maven 项目结构，1.5 小时，输出目录和 pom.xml。
  - 刷 2 道错题，1.5 小时，输出二刷记录。
- 周二：4 小时
  - 整理已有 Java 练习到 Maven 项目，2 小时，输出代码结构。
  - 刷 1 道错题，1 小时，输出二刷记录。
  - 更新算法追踪表，1 小时，输出错因变化。
- 周三：4 小时
  - 阅读 JUnit 基础，1 小时，输出要点清单。
  - 给计算器核心逻辑写测试，2 小时，输出测试代码。
  - 运行测试并记录结果，1 小时，输出测试记录。
- 周四：4 小时
  - 给文件读写模块写测试，2 小时，输出测试代码。
  - 刷 2 道复习题，1.5 小时，输出题解记录。
  - 写 Maven 笔记大纲，0.5 小时，输出初稿。
- 周五：4 小时
  - 完成 Maven/JUnit 笔记，1.5 小时，输出成稿。
  - 刷 2 道复习题，1.5 小时，输出题解记录。
  - 整理本周 5 个面试问题答案，1 小时，输出追踪记录。
- 周六：5 小时
  - 刷 3 道复习题，2 小时，输出题解记录。
  - 更新项目 README 运行和测试说明，1.5 小时，输出 README。
  - 提交 Git commit，1.5 小时，输出提交记录。
- 周日：4 小时
  - 复述 Maven 生命周期和 JUnit 用法，1 小时，输出文字稿。
  - 检查项目目录是否清晰，1 小时，输出改进清单。
  - 完成周复盘，1 小时，输出复盘文件。
  - 调整下周最小目标，1 小时，输出计划修正。

## 4. 学习输入

- Maven 官方文档：Getting Started、POM、Build Lifecycle。
- JUnit 5 官方 User Guide：Writing Tests、Assertions。
- JavaGuide：Maven 基础。
- 代码随想录：前 6 周错题对应章节。
- LeetCode：错题二刷。

## 5. 输出物

- 至少 10 道算法题记录。
- 1 篇 Maven/JUnit 笔记。
- 1 个 Maven 小项目结构。
- 至少 5 个 JUnit 测试用例。
- 1 个 README 更新。
- 1 次 Git commit。
- 1 次周复盘。

## 6. 验收标准

- 能运行 `mvn test` 并看到测试结果。
- 能解释 Maven 坐标、依赖、生命周期的含义。
- 能给一个纯函数写 JUnit 断言测试。
- 能通过 README 让别人运行项目和测试。
- 能复刷 10 道错题并标记仍然卡住的题。
