# 2026-W04：Java IO 与 Git/Linux 基础实践

日期：2026-07-13 至 2026-07-19

## 1. 本周唯一主目标

完成一个基于文件读写的小模块，并把 Git/Linux 基础操作变成可复现记录。

## 2. 本周任务

- 写 Java 文件读写、目录遍历、try-with-resources 练习。

- 完成 10 道字符串、双指针、数组复习题。
- 写 1 篇笔记：Java IO 基础和资源关闭。
- 整理 20 个 Git/Linux 高频命令并实际执行记录。

### 选做任务

- 给项目增加简单日志文件。
- 用 Git 分支完成一次 feature 开发再合并。

### 低状态保底任务

- 完成 3 道字符串或数组题。
- 写 1 篇短笔记：try-with-resources 做了什么。
- 做 1 次项目 README 或代码 commit。
- 完成 1 次周复盘。
