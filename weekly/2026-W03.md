# 2026-W03：Java 集合与栈队列题型

日期：2026-07-06 至 2026-07-12

## 1. 本周唯一主目标

用 Java 集合解决常见数据处理问题，并完成栈队列题型训练。

## 2. 本周任务

### 必做任务

- 写 ArrayList、LinkedList、HashMap、HashSet、Queue、Deque 的使用练习。
- 完成 10 道栈、队列、哈希、字符串基础题。
- 写 1 篇笔记：ArrayList、LinkedList、HashMap 的核心差异。
- 给命令行小项目增加历史记录或统计功能。
- 完成 1 次 Git commit 和 1 次周复盘。

### 选做任务

- 阅读 HashMap 扩容机制的 JavaGuide 说明。
- 手写一个简化版 LRU 缓存。

### 低状态保底任务

- 完成 3 道栈队列题。
- 写 1 篇短笔记：HashMap 为什么查询快。
- 做 1 次项目 commit。
- 完成 1 次周复盘。
