# 2026-W01：启动系统与 Java 基础闭环

日期：2026-06-22 至 2026-06-28

## 1. 本周唯一主目标

建立可持续执行的学习闭环，并用 Java 完成第一批可运行练习。

## 2. 本周任务

### 必做任务

- 建好本仓库的使用习惯：每天记录任务，周末完成复盘。
- 完成 JDK、IDE、Maven、Git 基础环境检查，记录环境版本。
- 写 8 个 Java 基础练习：变量、条件、循环、数组、方法、输入输出、类、简单异常。
- 完成 10 道数组/哈希基础算法题，并记录错因。
- 提交至少 1 次 Git commit。

### 选做任务

- 整理 20 个 Linux 高频命令及用途。
- 用 Java 写一个命令行计算器，并补充 README 运行方式。

### 低状态保底任务

- 完成 3 道数组题。
- 写 1 篇 300 到 500 字短笔记：JDK、JRE、JVM 区别。
- 做 1 次 README 或笔记 commit。
- 完成 1 次周复盘。
