# 年度路线图

周期：2026-06-22 至 2027-08-31。

本路线只定义阶段目标，不写死全年每日任务。详细计划只维护最近 4 到 8 周，每 4 周根据完成情况、招聘市场和项目进展滚动更新。

## 阶段 1：Java 基础 + 数据结构算法 + Git/Linux 基础

时间：2026-06-22 至 2026-08-31

阶段目标：

- 完成 Java 语法、集合、异常、IO、泛型、基础并发的可运行练习。
- 建立算法刷题节奏，覆盖数组、链表、栈、队列、哈希、字符串、二分、双指针。
- 能独立使用 Git 完成分支、提交、合并、回滚到指定提交的演示。
- 能在 Linux 环境完成文件、进程、网络、权限、日志查看等基础操作。
- 产出第一个 Java 命令行小项目，为后续 Spring Boot 项目做准备。

阶段产出：

- 至少 80 道算法题记录。
- 至少 12 篇基础笔记。
- 1 个 Java 基础项目 README。
- 每周一次复盘。

阶段产出：
