# 技能树

## 主线 A：Java 后端

- Java 基础：语法、集合、异常、IO、泛型、反射、注解。
- Java 并发：线程、线程池、锁、AQS、CAS、ThreadLocal、CompletableFuture。
- Spring：Spring Boot、Spring MVC、参数校验、统一异常、AOP、事务、拦截器。
- MySQL：表设计、索引、事务、锁、执行计划、慢 SQL。
- Redis：数据结构、缓存策略、分布式锁、持久化、淘汰策略。
- 工程能力：Git、Linux、Maven、Docker、Nginx、日志、测试、接口文档。

## 主线 B：计算机基础

- 算法：数组、链表、栈队列、哈希、字符串、二分、双指针、树、堆、动态规划、图。
- 网络：HTTP、HTTPS、TCP、UDP、DNS、Cookie/Session/JWT。
- 操作系统：进程线程、内存管理、文件系统、IO、锁、死锁。
- JVM：内存区域、类加载、GC、调优参数、线上问题排查。

## 主线 C：分布式与平台开发

- 分布式基础：CAP、BASE、一致性、分布式事务、限流熔断。
- MQ：异步削峰、重复消费、消息可靠性、顺序消息、死信队列。
- 平台能力：权限、审计、任务调度、状态机、日志追踪、成本统计。
- 稳定性：幂等、重试、超时、降级、压测、监控。

## 主线 D：AI 应用工程

- RAG：文档解析、切分、向量化、检索、重排、引用溯源、评测集。
- Agent：工具注册、任务规划、任务执行、状态机、人工确认、失败重试。
- 工程化：权限、日志、成本、评测、缓存、数据隔离、可观测性。
- 面试表达：业务场景、失败案例、效果指标、工程取舍。

## 优先级规则

- 2026-06 至 2026-12：Java 后端和计算机基础优先。
- 2027-01 至 2027-03：项目工程化和分布式优先。
- 2027-03 至 2027-05：RAG/Agent 与后端结合优先。
- 2027-06 至 2027-08：面试输出、二刷和简历优先。
