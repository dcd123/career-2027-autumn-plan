# 每周维护 Prompt

请基于当前仓库内容，帮我完成一次周维护。

注意：
1. 不要重新生成全年计划。
2. 不要增加任务量。
3. 不要要求我每天维护。
4. 只做本周复盘和下周计划。
5. 下周仍然只能有一个主目标。

请完成以下任务：

## 1. 检查本周完成情况

请查看：
- WEEKLY_CURRENT.md
- notes/weekly-notes/
- projects/
- interview/
- resume/
- review/

只根据仓库已有内容判断，不要凭空假设。

## 2. 生成本周复盘

保存到：

review/2026-Wxx-review.md

内容包括：
- 本周完成度评分
- 实际完成内容
- 未完成内容
- 最大问题
- 下周是否降级
- 可沉淀到简历 / 面试的内容

## 3. 归档本周计划

把当前 WEEKLY_CURRENT.md 归档到：

weekly/2026-Wxx.md

## 4. 生成本周学习笔记

保存到：

notes/weekly-notes/2026-Wxx.md

要求：
- 不要长篇摘抄
- 只整理最重要的内容
- 包含算法、基础知识、项目、面试沉淀

## 5. 生成下周 WEEKLY_CURRENT.md

要求：
- 只有一个主目标
- 必做任务不要超过 5 项
- 必须有低状态保底任务
- 必须有输出物
- 必须有验收标准
- 适合手机查看
- 不要安排每日详细任务
